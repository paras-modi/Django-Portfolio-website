Portfolio Website

Client: Self
Project Description: This is my portfolio website to rteflect my expertis & skills.This website contains several pages - homepage,project,resume ,contact. I used the Django MVT (Model View Template), a software design pattern. MVT is a collection of three important components Model View and Template. The Model helps to handle database. It is a data access layer which handles the data.The Template is a presentation layer which handles User Interface part completely. In Templates ,I used Bootstrap to make my pages more attractive.The View is used to execute the business logic and interact with a model to carry data and renders a template.
Technology Used: Django,Bootstrap,Html,Css,sql lite
Programming Language used: Python
Platform Used: Visual Studio Code
